What exaclty is an algorithm and why would you care?
That might be a very good question, and I hope I will be able to give you a good answer.

One definition often used to define Algorithms is: a set of steps to accomplish a task

For example: You are going to make a grilled cheese sandwich. To do so, you need to complete a couple of different steps:
Step 1: Acquire bread
Step 2: Apply cheese
Step 3: Grill!
Step 4: Enjoy!

The same logic is used with computer algorithms.
A Computer Algorithm is a set of steps for a computer program to accomplish a task.
